SVG/HTML theme Render

Custom web Component to display source of SVG with theme-able attributes

Use classes to place theme colors, generated on load as css variables:

.cf-c1, .cf-c2, etc, where 'cf' relates to 'fill', and '-c1' matches the color in the theme array.

similarly, .cs-c1, cs-c2... would related to stroke colors.

Usage

<areweave-viewer source="..." theme="..."></areweave-viewer>

Props

  • content: html string to be rendered
  • id (optional): arweave id. Will display as file
  • aspect (optional): aspect ration for displayed element (used with src or id to determine height) eg. 1/1
  • theme (optional): pipe or comma separated array of color values
  • hue (optional): hue value between 0 and 360. This will auto-generate a theme

Themes

  • Supply hex (with or without #), rgb, hsl values
  • values must be same type
  • Any number but should match the number of variables assigned in the artwork
  • Will be applied in order. So the first color will be var(--c-c1)

Examples

With no theme
