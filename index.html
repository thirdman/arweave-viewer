SVG/HTML theme Render

Custom web Component to display source of SVG with theme-able attributes

For demos run a local server and view /demos 

...
    source="https://arweave.net/4A97ZHcjZ5AOjwBCXktL6tzS4mV_MPJNR00UIYC3uHk"
    aspect="1/1"
    theme="#000000|#444444|#777777|#aaaaaa|#cccccc"
    speed="1"
    intensity="2"
